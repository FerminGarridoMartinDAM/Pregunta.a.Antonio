• Pseudoclases:enabled

    La pseudoclase :enabled nos permite recuperar cualquier elemento habilitado en el documento, diferenciando asi a los deshabilitados.

    Ejemplo Pseudoclases:enabled

    En este ejemplo vamos a recupera "cambiar" el elemento que esta habilitado o con el atributo enabled activo y lo vamos a poner de color verde y el disabled en rojo. He cambiado un poco el codigo añadiendole a las etiquetas button la clase ejemploenabled para que solo cambie los botones de este ejemplo y no la de las anteriores , por tanto tambien he añadido al codigo un selector de atributo class="ejemploenabled" para que no afecte a los botones de arriba. que se añade con [class="ejemploenabled"] button[class="ejemploenabled"]:disabled { color: red; } button[class="ejemploenabled"]:enabled { color: green; }
